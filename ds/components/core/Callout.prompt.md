Left-bordered aside for a key statement, definition, or caveat — the documentation system's callout.

```jsx
<Callout title="Bénéfice clé.">Le coût de spatialisation est fixe (6 convolutions).</Callout>
<Callout variant="muted">Seule la v0 est spécifiée finement.</Callout>
```

`variant`: `default` (black left border, important), `muted` (gray border, smaller, for asides), `inverse` (dark viewport). `title` renders a bold inline lead-in. No icons, no colored background — emphasis is the border.
