Uppercase, tracked monospace kicker placed above a heading — the brand's signature section label.

```jsx
<Eyebrow>Branche Cadrage</Eyebrow>
<h1>Le modèle du moteur</h1>
```

`tone`: `muted` (default gray), `ink` (black), `inverse` (for dark viewport). Always uppercase + wide tracking; pair it directly above an h1/h2.
