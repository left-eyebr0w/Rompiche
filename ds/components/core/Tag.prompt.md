Status / metadata pill in the brand's mono "technical voice" — uppercase, tracked, monochrome.

```jsx
<Tag variant="spec">spécifiée</Tag>
<Tag variant="tbd">TBD</Tag>
<Tag variant="solid" dot>v0.5</Tag>
```

Variants mirror the source documentation: `spec` (black border = decided/authoritative), `tbd` (dashed = open/placeholder), `solid` (filled ink chip), `default` (gray hairline). Set `inverse` on the dark viewport, `dot` for a leading status dot. Never colored — the whole point is monochrome status.
